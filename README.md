# AMQ Download App (WIP)

⚠️ **Work In Progress**: This project is currently under development.

A full-stack application in development for downloading songs from AnisongDB/Amq. The project features a React Material-UI (MUI) single-page application (SPA) on the front-end and an Express API for the back-end.

## Features (Planned)

- **Front-end**: 
  - React with Material-UI (MUI) for a modern, responsive UI.
  - Single-page application (SPA) for smooth user experience.
  - Interface for initiating and managing downloads from an external API.
  
- **Back-end**: 
  - Express API for handling data retrieval and download processing.
  - Provides RESTful endpoints for the front-end.

- **Planned**:
  - Support for multiple file formats and concurrent downloads.
  - User authentication (if needed).

## Technologies (In Progress)

- **Front-end**: React, Material-UI (MUI)
- **Back-end**: Node.js, Express
- **API Integration**: REST API for data retrieval

## Installation (WIP)

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/amq-download-app.git
    cd amq-download-app
    ```

2. Install back-end dependencies:
    ```bash
    cd backend
    npm install
    ```

3. Install front-end dependencies:
    ```bash
    cd frontend
    npm install
    ```

## Usage (WIP)

- Front-end and back-end will be run separately during development.
- Access the app at `http://localhost:3000` for the front-end once both servers are up.
